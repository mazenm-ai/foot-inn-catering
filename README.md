Food Inn Catering Website

Welcome to the code repository for the Food Inn Catering website!

This is a modern, fast, and mobile-friendly website built with React and Tailwind CSS, designed specifically to help capture catering leads in Ottawa.

🌟 Features

Mobile-First Design: Looks perfect on phones, tablets, and desktop computers.

Interactive Menu: Filterable food categories (Appetizers, Mains, Desserts, etc.).

Quote Request System: A built-in form for customers to request quotes for corporate events, weddings, and private parties.

Easy to Update: Menu items and images are organized so they can be easily swapped out.

🛠️ How to Edit the Content

All the main content for this website is located inside the App.jsx file.

To change the Menu: Open App.jsx, find the list called const menuItems = [...], and edit the text or add new items there.

To change Images: Open App.jsx, find any image tag (it looks like <img src="..." />), and replace the link inside the quotation marks with a link to your own photo.

To change Links: Find the href="..." attribute on buttons or links and replace the destination URL.
